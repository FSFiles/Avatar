# AI Creator Avatar Platform

An AI-powered platform that transforms a simple topic into a complete talking avatar video using automation and generative AI.

---

## Project Overview

AI Creator Avatar is an end-to-end content workflow platform that allows users to:

- Enter a topic  
- Generate a script using AI  
- Create voice narration  
- Generate a talking avatar video  
- Download and manage videos  

This platform simplifies the entire content creation process into a few clicks.

---

## Problem Statement

Creating video content requires:

- Script writing  
- Voice recording  
- Video editing  
- On-camera presence  

This process is time-consuming and difficult for many creators.

---

## Solution

Our platform automates the entire workflow:

Topic → Script → Voice → Avatar → Video

Users can generate professional content in seconds without technical skills.

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Supabase (Database, Authentication, Storage)

### AI Integration
- Google AI Studio (Gemini API)

### Video Generation
- D-ID API (Talking Avatar)

---

## Features

- AI Script Generation  
- Voice Generation  
- Avatar Customization  
- Talking Avatar Video Creation  
- MP4 Download  
- Recent Videos Dashboard  
- User Authentication  

---

## Workflow

1. User enters a topic  
2. AI generates a script  
3. User customizes avatar  
4. Voice is generated  
5. Avatar speaks with lip-sync  
6. Video is generated and saved  
7. User can download or view later
